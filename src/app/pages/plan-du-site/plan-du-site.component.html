Légende

  • Lien vers une page
  • Action (ouvrir, déclencher…)

Accueil

  • Accueil
    • Cliquer sur “Consulter les modèles” (redirige vers connexion)
    • Cliquer sur “Gérer les modèles” (redirige vers connexion)
    • Cliquer sur “Historique des conventions” (redirige vers connexion)

Authentification

  • Se connecter
    • S’authentifier avec son e-mail et son mot de passe
    • Aller sur la page mot de passe oublié
    • Aller sur la page d’inscription
  • S’enregistrer
    • Créer un compte utilisateur GenioService
    • Aller sur la page de connexion
  • Mot de passe oublié
    • Envoyer un lien de réinitialisation de mot de passe
    • Retourner à la page de connexion
  • Réinitialiser le mot de passe
    • Changer son mot de passe
    • Retourner à la page de connexion

Rôle : Consultant

    • Rechercher un modèle par nom ou année
    • Cliquer sur l’icône en forme d’œil pour voir les détails du modèle

Rôle : Exploitant

    • Rechercher un modèle par nom ou année
    • Cliquer sur l’icône en forme d’œil pour voir les détails du modèle
    • Rechercher une convention par nom de l’étudiant, année ou promotion
    • Cliquer sur l’icône œil pour afficher les détails d’une erreur

Rôle : Gestionnaire

    • Rechercher un modèle par nom ou année
    • Cliquer sur l’icône en forme d’œil pour voir les détails du modèle
    • Rechercher une convention par nom d’étudiant, année ou promotion
    • Cliquer sur l’icône en forme d’œil pour afficher les détails d’une erreur
    • Rechercher un utilisateur par e-mail, rôle ou état du compte
    • Modifier les informations d’un utilisateur (rôle, activation)
    • Supprimer un utilisateur avec une confirmation via une fenêtre modale
    • Cliquer sur l’icône cloche pour consulter les nouvelles demandes d’activation
    • Voir un message du type “Vous avez 1 demande d’habilitation” ou “Aucune nouvelle demande”
    • Ajouter un nouveau modèle en important un fichier .docx
    • Dans l’onglet "Modifier", cliquer sur l’icône crayon dans le tableau pour ouvrir une fenêtre modale de modification
    • Dans l’onglet "Archiver", cliquer sur l’icône d'archivage dans le tableau pour afficher une fenêtre de confirmation
    • Rechercher un modèle par titre ou année dans les zones "Modifier" ou "Archiver"
